	- NHibernate & FluentNhibernate

	- AutoMapper
	- Log4Net
  - Client side
    - Twitter Bootstrap
	- Durandaljs & Transitions

	- jQuery plug-ins
	  - jQuery validation
	  - jTable (TODO: add with extension)
	  - toastr
	  - spin.js
